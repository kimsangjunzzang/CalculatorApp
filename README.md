# iOS Calculator

iOS 기본 계산기

## 1. 프로젝트 소개

SwiftUI를 이용해 iOS 기본 계산기 어플리케이션을 구현한 토이 프로젝트입니다.

## 2. 언어 및 프레임워크

language : Swift
Framework : SwiftUI, Foundation

## 3. 주요 기능

- 기본적인 사칙 연산
- 연산 ( +, - , *, / ) 이 섞여 있을경우 우선순위를 바탕으로 연산
- Percnet 기능
- 초기화 기능
- opposite 기능
